# Product Overview

React Tetris is a fully functional browser-based Tetris game built with React. It implements classic Tetris gameplay with modern UI and complete game mechanics.

## Core Features

- 7 classic tetrominoes (I, O, T, S, Z, J, L) with distinct colors
- Keyboard controls: arrow keys for movement/rotation, space for hard drop
- Line clearing with progressive difficulty (speed increases every 10 lines)
- Score and level tracking system
- Next piece preview
- Pause/resume functionality
- Game over state with statistics
- Responsive design for desktop, tablet, and mobile

## Scoring System

- 1 line: 100 × level
- 2 lines: 300 × level
- 3 lines: 500 × level
- 4 lines (Tetris): 800 × level

Level increases every 10 lines cleared, with drop speed accelerating accordingly.
